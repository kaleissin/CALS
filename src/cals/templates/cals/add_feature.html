Add a missing feature

Send an email to taliesin+calsfeature@nvg.org with the following contents:

  1. On the first line: the feature-number from WALS.
  2. On subsequent lines: the values, as spelled at WALS, one per line.

An example follows:

1
Small
Moderately small
Average
Moderately large
Large

You can put several features in the same mail, like so:

1
Small
Moderately small
Average
Moderately large
Large
19
None
Clicks
Labial-velars
Pharyngeals
'Th' sounds
Clicks, pharyngeals, and 'th'
Pharyngeals and 'th'

I'll read over it and batch-convert it into something the database can import, and add the name of the first feature-sender to the thanks-page.
